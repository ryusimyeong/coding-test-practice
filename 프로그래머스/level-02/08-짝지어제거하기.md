
스택을 활용했으나 시간 초과
```js
const solution = s => {
    let stack = [];
    let sArr = s.split("");
    
    while (sArr.length > 0) {
        if (stack[stack.length-1] !== sArr[0]) {
            const letter = sArr.shift();
            stack.push(letter);
        } else {
            sArr.shift();
            stack.pop();
        }
        if (stack.length > sArr.length) {
            break;
        }
    }
    
    return stack.length === 0 ? 1 : 0;
};
```
while 활용. 위의 풀이보다는 빨랐지만 시간 초과
```js
const solution = s => {
    let answer = 0;
    const arr = s.split("");
    let i = 0;
    while (arr.length > 1) {
        if (i >= arr.length) {
            break;
        }
        if (arr[i] === arr[i+1]) {
            arr.splice(i, 2);
            i = i - 1;
        } else {
            i++;    
        }
    }
    answer = arr.length === 0 ? 1 : 0;
    return answer;
};
```
